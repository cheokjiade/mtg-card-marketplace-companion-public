# MTG Marketplace Companion

A personal Chrome extension that builds a unified, searchable history of Magic: The Gathering card purchases across the marketplaces you already shop on. Captures order data passively as you browse, normalizes every line item against Scryfall, and gives you stats, a review queue, a cross-marketplace cart optimizer, and optional Google Drive sync between devices.

**Supported marketplaces**: TCGPlayer, Manapool, TheTCGMarketplace, Card Kingdom, Star City Games, Games Haven.
